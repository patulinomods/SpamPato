# SpamPato v3.0
Fazemos Spam com: **Uber**, **99**, **Aiqfome**, **RecargaMulti**, **Vivo** e **Claro**.

## Requisitos
* NodeJS Instalado

## Instalação
* ```apt update && apt upgrade```
* ```apt install git -y```
* ```git clone https://github.com/patulinomods/SpamPatoV3```
* ```cd SpamPatoV3```
* ```chmod +x *```
* ```bash install.sh```
Start: ```node spam.js```
